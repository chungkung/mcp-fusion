## Reporting Issues

When reporting a bug, please include:

1. **MCP Fusion version** (Settings > About)
2. **Operating system** and version
3. **Steps to reproduce** the issue
4. **Relevant log files** from the `logs/` directory
5. **Screenshots** or screen recordings if applicable

Use the **[Bug Report](https://github.com/chungkung/mcp-fusion/issues/new?template=bug_report.md)** template for the fastest response.

## Security Vulnerabilities
